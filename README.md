## Deploy to GitHub Pages

This repository is configured to deploy using the `gh-pages` branch.

Deploy (builds and publishes `build/` to the `gh-pages` branch):

```bash
npm run deploy
```

GitHub Pages settings (one-time):

- Repo → Settings → Pages
- Source: “Deploy from a branch”
- Branch: `gh-pages` / Folder: `/ (root)`

### Notes

- The deployment path is controlled by the `homepage` field in `package.json`.
